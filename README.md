Food on the Way
=====================
A web app which which displays a number of restaurants along the route to a destination.  
Users can save addresses for later use and filter the displayed restaurants.  
Utilizes the MEAN stack.

## Installation

npm install  

## Running the Server

mongod  
npm start
at 127.0.0.1:8000

## Testing

npm test
